STU-Check

ระบบ AI ประเมินผลการจัดการเรียนการสอนในห้องเรียน ใช้ Camera Webcam ดักจับภาพ แล้วส่งผ่าน HTTPS ไปยัง FastAPI Server ที่บ้านเพื่อวิเคราะห์อารมณ์ด้วย AI โดยส่งเพียง 1 ภาพ ต่อ 1 นาที เพื่อประหยัดแบนด์วิธ

โรงเรียน: พะเยาพิทยาคม สังกัดสำนักงานเขตพื้นที่การศึกษามัธยมศึกษาพะเยา
ผู้พัฒนา: นายสากล จันทร์ขาว, นายปริพัฒน์ มาแก้ว, นางสาวณิชา วันดู, นางสาวฐิติวรดา งานดี
ครูที่ปรึกษา: นายเอกพงษ์ แก่นจันทร์
สถาปัตยกรรม: Frontend (GitHub Pages) + Backend (FastAPI + Ngrok)
ความเป็นส่วนตัว: ไม่บันทึกวิดีโอ ส่งเพียงภาพนิ่ง 1 ภาพ/นาที และเก็บข้อมูลเป็นตัวเลขเท่านั้น
การเชื่อมต่อ: HTTPS ผ่าน Ngrok Tunnel ไปยัง Python FastAPI Server ที่คอมบ้าน
